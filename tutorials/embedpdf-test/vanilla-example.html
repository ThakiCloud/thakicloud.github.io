💡 구현 가이드

실제 EmbedPDF 구현 시에는 다음과 같은 코드를 사용합니다:

// EmbedPDF 초기화
import EmbedPDF from '@embedpdf/core';

const viewer = new EmbedPDF('#pdf-container', {
  plugins: ['annotations', 'search', 'zoom'],
  theme: 'light',
  toolbar: true
});

// PDF 로드
await viewer.load(pdfFile);

// 이벤트 리스너
viewer.on('pageChange', (page) => {
  console.log('현재 페이지:', page);
});
